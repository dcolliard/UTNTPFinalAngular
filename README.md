## 🚀 ¿Cómo correr el proyecto localmente?

1. **Clonar el repositorio**

   ```bash
   git clone https://github.com/tu-usuario/BikeStoreApp.git
   ```
2. **Instalar dependencias**

   ```bash
   npm install
   ```

3. **Ejecutar servidor**
  
   ```bash
   npm run dev
   ```

4. **Acceder en navegador**

     http://localhost:4200
